## Key finding from literature positioning

The closest classical analog is **Fisher sufficiency** (1920s) — the factorization theorem characterizing when a statistic is sufficient for a parameter. The Crossing Lemma is structurally a finite-combinatorial two-structure analog.

**Other neighbors to watch:**
- Optimal experimental design (Kiefer, Fedorov) — the applied instances (sensor placement, assay design, tomography) sit in this tradition.
- Gelfand pairs and Tannakian reconstruction.
- Galois theory / monodromy reconstruction.
- Matroid representability.

The **residual novelty** would be:
1. Binary sufficiency without probabilistic framework.
2. Productive Incompleteness addendum (failed-sufficiency as refinement, not useless).
3. Cross-domain applicability with a uniform statement.

**If** the lemma's precise statement resists translation into Fisher-sufficiency language **and** its applied instances produce discriminating predictions (not matched by classical OED), the novelty is strong. Both conditions are plausible but require the UOP arc content to verify.
