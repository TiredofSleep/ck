## Summary

We submit for consideration in JCAP a single-field quintessence model
defined by a logarithmic potential acting on a dimensionless scalar
field Ξ ∈ [0,1] with information-theoretic motivation. The model
contains two free parameters and produces:

- An exact analytic minimum at Ξ₀ = e⁻¹, fixing the cosmological
  constant scale to Λ ≈ 1.7 meV without anthropic input.
- A best-fit reduced-chi-squared χ² = 1.523 against DESI BAO + Planck
  CMB + Pantheon+ SNe (compared with χ² = 15.26 for ΛCDM at the same
  data combination), with parameter values Λ⁴/ρ_{c,0} = 0.231 and
  Ξ_i = 0.925 from the documented optimization in
  desi_xi_optimize_v2.py.
- Phantom-crossing equation-of-state w(z): w₀ = -0.7933,
  w_a = -0.4497, with the freezing branch confined to w(z) → -1 as
  z → -1 (asymptotic vacuum endpoint, not a transient "tracker").
- A bare-vs-renormalized vacuum contribution discussed explicitly
  in §4.1.

## Why JCAP

The model addresses two of the standard motivations for JCAP papers:
(i) it provides a falsifiable single-parameter alternative to the
cosmological constant compatible with all current cosmological probes;
(ii) the logarithmic potential is derived from an information-theoretic
counting argument (§2.1) rather than postulated, making the parameter
choice non-arbitrary. The primary observational tests are
straightforward extensions of existing DESI / Euclid / Roman pipelines.
