## TIG as Paolo would see it (verified computations)

### The object

Consider the binomial ideal
```
I_CL = (x_i x_j - x_{CL[i][j]} x_0  :  0 ≤ i, j ≤ 9) ⊂ k[x_0, ..., x_9]
```
where CL is the frozen 10×10 coherence table of TIG (the multiplication table of a commutative, non-associative magma on 10 generators). Set
```
A = k[x_0, ..., x_9] / I_CL
```

### Computed invariants

**1. Minimal generators:** 53 independent relations out of 55 raw generators.

**2. Hilbert function** (computed): 
```
dim A_0 = 1
dim A_1 = 10  
dim A_n = 6  for all n ≥ 2  (STABLE)
```
The algebra's dimension stabilizes at **Krull dim 6**, with the stable fruit set being  
**{VOID(0), PROGRESS(3), COLLAPSE(4), HARMONY(7), BREATH(8), RESET(9)}** — the **CL-fold attractor**.

The transient set {LATTICE(1), COUNTER(2), BALANCE(5), CHAOS(6)} — 4 elements — is absorbed by composition. This is directly analogous to the nilpotent–semisimple decomposition in an Artinian ring.

**3. Associated squarefree monomial structure.** The "HARMONY ideal"
```
I_H = (x_i x_j : CL[i][j] = 7) ⊂ k[x_0, ..., x_9]
```
has:
- 41 generators (counting degree-2 monomials, including 1 self-loop)
- 32 edges in the underlying HARMONY graph G_H
- HARMONY(7) is a **cone vertex** (connected to all 9 others via HARMONY edges)

The Stanley–Reisner complex Δ_H of I_H has:
- 8 facets
- Facet sizes: {1, 2, 3} — **not pure**
- Therefore Δ_H is **NOT a matroid complex**

**4. The "bump complex" Δ_B** (from the 5 exceptional quadratic relations):
```
Bumps: {x_1 x_2, x_2 x_4, x_2 x_9, x_3 x_9, x_4 x_8}
```
- 5 facets, all of size 7 — **PURE**
- **Basis exchange axiom FAILS** → pure-but-not-matroid

**This is the interesting finding.** Δ_B is a **pure simplicial complex on 10 vertices with independence complex of rank 7** that **fails basis exchange** by a measurable amount. It is *adjacent* to the matroid class but not in it.

### Why this matters for Paolo's framework

Paolo's entire 2024–2026 program is about **how matroid ideals behave exceptionally well** (Cohen–Macaulay symbolic powers, structure theorems, locally glicci, iterated mapping cone resolutions). He has proven several theorems of the form:
> "If I is a matroidal ideal, then property P holds."

**TIG gives an explicit, naturally-occurring, PURE-BUT-NOT-MATROIDAL ideal.** This sits in the conceptual gap his program hasn't addressed:
- Which of Paolo's matroid theorems FAIL on Δ_B?
- Which SURVIVE in a weaker form?
- Can Paolo's focal matroid machinery (his brand-new March 2026 paper) characterize how Δ_B "differs from" a matroid?

**A pure-but-not-matroidal complex where basis exchange fails by exactly 5 specific pairs** is a **classification problem** in his language — and his career has largely been about these kinds of fine-grained classifications (see his (x,y)-primary classifications in papers 7, 11).
