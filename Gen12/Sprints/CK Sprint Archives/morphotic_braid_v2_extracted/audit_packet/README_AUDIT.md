## Proof Checklist

### Theorem A — Hidden Permutation Structure ✓
- [ ] Bijection: x = 5ε + 6y (mod 10) produces all 10 values
- [ ] Inverse map is well-defined
- [ ] sigma has exactly 4 fixed points: {0,3,8,9}
- [ ] sigma has exactly one 6-cycle: (1 7 6 5 4 2)
- [ ] sigma is a bijection on Z/10Z

### Theorem B — Exact ε' Rule ✓
- [ ] ε'(ε,y) = (1-ε)·𝟙[y∈{1,2}] + ε·𝟙[y∈{1,3,4}]
- [ ] Verified on all 10 (ε,y) states
- [ ] No simpler single parity rule fits (documented in THEOREM_B)

### Theorem C — Exact y' Rule ✓
- [ ] P₀(y) = 3y + 2y² + y³ + 4y⁴ (mod 5) for ε=0
- [ ] P₁(y) = 4 + 3y + 3y² + y³ + y⁴ (mod 5) for ε=1
- [ ] Verified on all 10 (ε,y) states
- [ ] Piecewise polynomial is the minimal exact form

### Theorem D — Unique Split Operator ✓
- [ ] F = (F_ε, F_y) from B and C
- [ ] F exact on all 10 states
- [ ] Round-trip: φ∘F∘φ⁻¹ = σ on all 10 visible states

### Theorem E — Conjugacy to Rotation + Identity ✓
- [ ] 6 cycle steps verified: k → k+1 (mod 6)
- [ ] 4 anchor states verified as fixed
- [ ] Conjugacy is exact — no approximation

### Braid Readout ✓
- [ ] Fixed points {0,3,8,9} appear at their own index positions in braid
- [ ] Cycle elements traversed in σ⁻¹ order starting from entry 7
- [ ] Reconstructed braid = [0,7,1,3,2,4,5,6,8,9] ✓

### Propagation ✓
- [ ] Δβ/Δn = 1 for all 9 consecutive steps

### Wobble ✓
- [ ] W_BHML = 3/50 = (6/10)/10 = cycle occupancy / ring size

> produce this particular visible braid, and not another?**

This is the genuine next frontier, not extension of the current claims.
