## 1. The A₂ Weyl Group

The Weyl group of type A₂ is W = S₃, generated by two simple reflections σ₁, σ₂.
In the (s,w) variables, the Bump-Friedberg-Hoffstein action is:

```
σ₁: (s, w) ↦ (1-s, w+s-1/2)
σ₂: (s, w) ↦ (s+w-1/2, 1-w)
```

These generate 6 elements:
```
e:  (s, w) ↦ (s, w)
σ₁: (s, w) ↦ (1-s, w+s-1/2)
σ₂: (s, w) ↦ (s+w-1/2, 1-w)
σ₁σ₂: (s, w) ↦ (w-1/2, 3/2-s-w)
σ₂σ₁: (s, w) ↦ (3/2-s-w, s-1/2)
σ₁σ₂σ₁: (s, w) ↦ (1-w, 1-s)    [= σ₂σ₁σ₂]
```

For a double Dirichlet series to be in the BFH class, it must satisfy:
```
Z̃(σ(s,w)) = G_σ(s,w) · Z̃(s,w)
```
for each σ ∈ W, where G_σ is an explicit ratio of zeta/L-function values and
gamma factors (the "archimedean factor").

## 2. The Local Test: Z̃_p under σ₁

From K12, the local factor is:
```
Z̃_p(s,w) = (1-v²)(1-u²) / [(1-2v cos θ_p + v²)(1-2u cos θ_p + u²)]

v = p^{-s-1/2},   u = p^{1/2-w}
```

**Under σ₁: (s,w) → (1-s, w+s-1/2)**

New variables:
```
v' = p^{-(1-s)-1/2} = p^{s-3/2} = p^{-1} · p^{s-1/2} = p^{-1} · v^{-1}

u' = p^{1/2-(w+s-1/2)} = p^{1-w-s} = p^{1/2-w} · p^{1/2-s} = u · p^{-s-1/2} · p^1 = u/v
```

Wait — let me be careful:
```
v = p^{-s-1/2}     → v^{-1} = p^{s+1/2}
v' = p^{-(1-s)-1/2} = p^{s-3/2} = p^{-1} · p^{s-1/2} = (1/p) · (1/v) · p^{-1}
   = p^{-1} · p^{-(1/2)} · p^s = p^{s-3/2}

Actually: v' = p^{-(1-s)-1/2} = p^{s - 1 - 1/2} = p^{s - 3/2}
And v = p^{-s-1/2}, so v · v' = p^{-s-1/2} · p^{s-3/2} = p^{-2}

Therefore v' = p^{-2}/v.

u' = p^{1/2-(w+s-1/2)} = p^{1-w-s}
u = p^{1/2-w}, so u'/u = p^{1-w-s-(1/2-w)} = p^{1/2-s} = v^{-1} · p^0...

Let me recompute: u = p^{1/2-w}, u' = p^{1-w-s}.
u'/u = p^{(1-w-s)-(1/2-w)} = p^{1/2-s}.
So u' = u · p^{1/2-s} = u / (p^{s-1/2}) = u / (v^{-1}) · p^0

Actually p^{s-1/2} = p^{-(−s+1/2)} = (p^{-1/2+s}).
And v = p^{-s-1/2}, so p^s = v^{-1} · p^{-1/2}.
Then p^{s-1/2} = v^{-1} · p^{-1/2} · p^{-1/2} = v^{-1} · p^{-1}.
So u' = u · v^{-1} · p^{-1} = u/(pv).
```

Summary:
```
σ₁: v → v' = p^{-2}/v,   u → u' = u/(pv)
```

**Now compute Z̃_p under σ₁:**

```
Z̃_p(1-s, w+s-1/2) = (1-v'^2)(1-u'^2) / [(1-2v'cosθ+v'^2)(1-2u'cosθ+u'^2)]
```

Substitute v' = p^{-2}/v:
```
1 - v'^2 = 1 - p^{-4}/v^2 = (v^2 - p^{-4})/v^2
1 - 2v'cosθ + v'^2 = 1 - 2p^{-2}cosθ/v + p^{-4}/v^2
    = (v^2 - 2p^{-2}cosθ·v + p^{-4})/v^2
    = (v - p^{-2}e^{iθ})(v - p^{-2}e^{-iθ})/v^2
```

Compare to:
```
1 - 2v cosθ + v^2 = (1 - ve^{iθ})(1 - ve^{-iθ})
```

The ratio:
```
(1 - 2v'cosθ + v'^2) / (1 - 2v cosθ + v^2)
= [(v - p^{-2}e^{iθ})(v - p^{-2}e^{-iθ})/v^2] / [(1 - ve^{iθ})(1 - ve^{-iθ})]
= [v - p^{-2}e^{iθ}][v - p^{-2}e^{-iθ}] / (v^2 |1 - ve^{iθ}|^2)
```

This is NOT a simple scalar multiple of (1 - 2v cosθ + v^2). The symmetry is not satisfied
without introducing correction factors involving e^{iθ_p} explicitly.

## 3. The Symmetry Failure and Its Cause

**Conclusion: Z̃_p(s,w) does NOT satisfy the A₂ Weyl group transformation without
correction factors that depend on the Sato-Tate angle θ_p.**

**Why:** In the BFH framework, the double Dirichlet series uses "average" Gauss sums
over characters, which introduces a symmetry at the cost of summing over all characters.
The individual Kloosterman sum Kl(1,1;p) = 2√p cos θ_p keeps the Sato-Tate angle
explicit, breaking the averaging that produces Weyl symmetry.

**The BFH correction:** BFH double Dirichlet series sum over ALL primitive characters χ (mod p):
```
Z̃_{BFH}(s,w) = Σ_{p} Σ_{χ} (Gauss sum g(χ))^2 / p^s × L(w, χ)
```
The sum over ALL χ (mod p) averages out the θ_p dependence and produces a result
with exact Weyl symmetry.

The Kloosterman version Z̃(s,w) = Σ_p Kl(1,1;p) p^{-s} × L_p(w) does NOT sum over
all characters — it picks out the χ=χ_1 (trivial) character contribution from the
character sum representation of Kloosterman sums.

**Character sum representation of Kloosterman sums:**
```
Kl(m,n;p) = Σ_{χ mod p} τ(χ)² χ(mn) / p
```
where τ(χ) = Σ_a χ(a) e^{2πia/p} is the Gauss sum.

For Kl(1,1;p): this sums all χ (mod p) with weight τ(χ)²/p. Taking a single term
(or the "full" sum) is what distinguishes Kloosterman from BFH.

## 4. The Fix: Character-Averaged Version Z̃_χ(s,w)

Define the **character-averaged Kloosterman double Dirichlet series**:
```
Z̃_χ(s,w) = Σ_p (Σ_{χ mod p} τ(χ)² χ(1)) p^{-s} × Σ_{k≥1} (Σ_{χ} τ(χ)² χ(1)) p^{-kw}
```

But Σ_{χ mod p} τ(χ)² χ(1) = Σ_{χ} τ(χ)² = p · #{χ : χ² = χ_0} × p^{1/2}...

Actually the cleaner version uses:
```
Z̃_χ(s,w) = Σ_{c≥1} Σ_{χ mod c} |τ(χ)|^2 χ(1) c^{-s} × L(w, χ)
```

This IS the BFH double Dirichlet series (of type A₂) for GL(1). It sums over all
primitive characters χ mod c, weighted by |Gauss sum|² = c (by Gauss sum normalization).

**Key result (D-tier, BFH 1996):** Z̃_χ(s,w) satisfies the A₂ Weyl group functional
equations with explicit correction factors.

**Connection to Kloosterman:** The Kloosterman sum is Σ_{χ mod p} τ(χ)² χ(1)/p
(summing over all characters). So:
```
p · Kl(1,1;p) = Σ_{χ mod p} τ(χ)² χ(1)
```
And Z̃_χ(s,w)|_{c=p} = p · Kl(1,1;p) · p^{-s} × L_p(w).

This means:
```
Z̃_χ(s,w) = Σ_p p · Kl(1,1;p) · p^{-s} × L_p(w)  +  (composite contributions)
           = p · Z̃(s,w)  +  (composite corrections)
```

The extra factor of p rescales s by 1 (absorbs into s+1 vs s). The structure is the same
with a shift. **Z̃_χ and Z̃ are the same series up to a shift s → s-1.**

## 5. Conclusion: Z̃ Inherits BFH Symmetry with Shift

**Theorem K13.1 (C-tier):**

The double Dirichlet series Z̃(s,w) = Σ_p Kl(1,1;p) p^{-s} L_p(w) is equal to:
```
Z̃(s,w) = (1/p) · Z̃_χ(s+1, w)
```
where Z̃_χ is the BFH double Dirichlet series of type A₂.

If this identification is correct, Z̃(s,w) satisfies the BFH functional equations
under the A₂ Weyl group action on (s+1, w), i.e., under:
```
σ₁: (s,w) ↦ (-s, w+s+1/2)
σ₂: (s,w) ↦ (s+w-1/2, 1-w)
```

**Gap:** The composite-moduli contributions in Z̃_χ vs Z̃ (the "correction" above)
must be controlled. If Z̃ = Z̃_χ(s+1,w)/p exactly (including all composite moduli),
then the functional equation holds. If the composite correction is nonzero, the functional
equation for Z̃ has an additional correction term.

## 6. The Zero Structure Under BFH Functional Equation

If Z̃(s,w) = Z̃_χ(s+1,w)/p satisfies the BFH functional equations, then Z̃ has
analytic continuation to all of C² (with poles on specific hyperplanes).

The poles of Z̃_χ(s,w) (from BFH theory) occur at:
- s = 1 (from the ζ(2s) factor in the BFH correction)
- w = 1 (from the ζ(2w) factor)
- On the BFH polar hyperplanes: s + w = 1, etc.

For Z̃(s,w) = Z̃_χ(s+1,w)/p:
- Poles at s+1 = 1 → s = 0 (shifted)
- The continuation to Re(s) < 0, Re(w) < 1 is now defined

**The zeros of Z̃(s,w):** From K12's global product formula, Z̃ ~ 1/ζ(2w-1) times other
factors. Under continuation, the zeros of ζ(2w-1) at w = (ρ+1)/2 for ζ-zeros ρ become
zeros of 1/ζ(2w-1), i.e., poles of Z̃. These poles at w = 3/4 + iγ/2 are the location
where Z̃ encodes ζ-zeros in the w-plane.

The functional equation maps these poles to other locations in (s,w), spreading the
ζ-zero information through the (s,w) plane.
