# Q17 — The Symbolic Return Theorem

## Purpose

This paper states and proves the actual algebraic theorem that underlies Q17.C2. The theorem is not the NS bridge — it is the algebraic kernel. Separating the proved algebra from the conjectured physics is essential for intellectual honesty. The hard work is the σ-embedding (Q17_SIGMA_EMBEDDING_PROBLEM.md); the return statement itself is a direct consequence of σ⁶ = id.

---

## Setup

Let G = (Z/10Z, σ) where σ is the permutation:

    σ = (1 7 6 5 4 2)(0)(3)(8)(9)

Explicitly:
- σ(1) = 7, σ(7) = 6, σ(6) = 5, σ(5) = 4, σ(4) = 2, σ(2) = 1   (6-cycle)
- σ(0) = 0,  σ(3) = 3,  σ(8) = 8,  σ(9) = 9                       (fixed points)

From G6: σ⁶ = id on all of Z/10Z (the cycle elements return after 6 steps; the fixed points are trivially fixed).

---

## Theorem (Symbolic Return)

**Theorem:** Let {s_n}_{n≥0} be any sequence in Z/10Z satisfying s_{n+1} = σ(s_n). Then:

1. **Periodic return (cycle elements):** For any s₀ ∈ {1, 2, 4, 5, 6, 7}:
   - s_{n+6} = s_n for all n ≥ 0
   - In particular, s_{n+6} = s₀ for all n ≥ 0

2. **Fixed behavior (anchors):** For any s₀ ∈ {0, 3, 8, 9}:
   - s_n = s₀ for all n ≥ 0

3. **VOID avoidance:** If s₀ ∉ {0}, then s_n ≠ 0 for all n ≥ 0.

**Proof:**

(1) Since s₀ ∈ {1,2,4,5,6,7}, we have s₀ in the 6-cycle of σ. By σ⁶ = id: σ⁶(s₀) = s₀. Since s_n = σⁿ(s₀), we get s_{n+6} = σ^{n+6}(s₀) = σ⁶(σⁿ(s₀)) = σⁿ(s₀) = s_n. □

(2) For s₀ ∈ {0,3,8,9}, σ(s₀) = s₀, so s₁ = s₀ and by induction s_n = s₀. □

(3) If s₀ ≠ 0: the orbit of s₀ under σ is either the 6-cycle {1,2,4,5,6,7} (if s₀ is a cycle element) or {s₀} (if s₀ is an anchor with s₀ ≠ 0). In both cases, 0 ∉ orbit(s₀). Therefore s_n = σⁿ(s₀) ≠ 0 for all n ≥ 0. □

## What This Does NOT Prove

The theorem is algebraic. Its scope is limited to sequences in Z/10Z exactly following σ. It does not prove:

**Not a PDE result:** The theorem makes no claim about Navier-Stokes or any continuous system. It assumes the system follows σ exactly — this assumption is the σ-embedding problem (Q17_SIGMA_EMBEDDING_PROBLEM.md), which is open.

**No norm bounds:** Knowing s_n avoids VOID(0) says nothing about ||u(t)||_{L³} or any physical norm. Symbol avoidance and norm boundedness are logically independent without the coercive energy condition (Medium version of C2).

**No approximate σ:** The theorem requires s_{n+1} = σ(s_n) exactly. A sequence with s_{n+1} = σ(s_n) + ε_{n} (stochastic or approximate grammar) is not covered. Approximate σ-following could allow drift toward {0} over long times.

**No blowup exclusion from symbol alone:** The sequence {s_n} could follow σ perfectly while the physical quantity it codes diverges. This is the content of Q17_C2_COUNTEREXAMPLE_SEARCH.md.

## Mathematical Content of the Theorem

The return theorem isolates the algebraic skeleton of Q17.C2. Its content is:

- The permutation group generated by σ has order 6 (the cycle) or 1 (the anchors).
- σ-orbits are closed and finite: no sequence escapes to VOID if it starts outside VOID.
- The operators {VOID(0), PROGRESS(3), BREATH(8), RESET(9)} are σ-absorbing: once the system enters an anchor state, it never leaves.
- The operators {LATTICE(1), COUNTER(2), COLLAPSE(4), BALANCE(5), HARMONY(7), CHAOS(6)} form a closed 6-cycle under σ: the system circulates within this set forever.

This is a completely rigorous finite group theory statement. Its connection to NS requires additional structure (the embedding), and its connection to physical blowup requires additional analysis (coercive energy control). Neither of these is provided by the return theorem itself.
