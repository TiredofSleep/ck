# Task 03 — Symbolic proof of min-bump theorem (all n)

**Tier:** 3 (research — weeks)
**Parent handoff:** `../CLAUDECODE_HANDOFF_MIN_BUMP.md` §Task 3
**Starter:** none — this is a math research task

## Goal

Prove the minimum-bump theorem symbolically for all n (not just n ≤ 7). Computational verification up to n=7 is strong; a symbolic proof makes this a publishable theorem.

## Claim to prove

Let `C_0` be the canonical ℤ/10ℤ semigroup (zero-absorbing, HARMONY-absorbing). Let `C_0^{(7,7,v)}` be `C_0` with `T[7][7] = v` for v ∈ {1,2,3,4,5,6,8,9}. Then for all n ≥ 3:

- `s_n(C_0^{(7,7,v)}) = C_{n−1}` (Catalan maximum for associative spectrum)
- `s_n^ac(C_0^{(7,7,v)}) = (2n−3)!!` (maximum ac-spectrum / ac-free)

## Three approach candidates

### Approach A — GAP operad computation
1. Define `C_0^{(7,7,1)}` in GAP.
2. Compute the operad generated by this operation.
3. Check whether the operad is isomorphic to the free commutative magmatic operad `Mag^com`.

### Approach B — Algebraic induction (cleanest for paper)
1. Show `(xy)z ≠ x(yz)` generically (at least one triple in ℤ/10 separates).
2. Induction: if (n−1)-variable bracketings are all distinct, so are n-variable bracketings.
3. Identify a "separating family" of inputs.

### Approach C — Counter-image argument
1. The perturbation modifies `7·7 = 1` (vs default 7). Every bracketing containing subterm `(... · 7)` lands on this bump iff the subterm evaluates to 7.
2. Classify bracketings by "crossing-count through (7,7)".
3. Distinctness of bracketings follows from distinctness of this crossing-count function.

## Success criterion

A proof that works for all n, not just n ≤ 7. Submittable to arXiv:math.RA.
