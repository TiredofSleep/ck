## Abstract

We prove that the sequence **0713245689** — the coherence ordering of the
morphotic lattice (TIG) — is generated by a permutation on Z/2 × Z/5 that is
exactly conjugate to the simplest possible dynamics: **identity on four fixed
points and cyclic rotation k → k+1 (mod 6) on a single 6-cycle.**

The visible sequence is a readout of this operator via the bijection x = 5ε + 6y
(mod 10). The apparent algebraic complexity of the split operator is entirely an
artifact of the coordinate encoding. The underlying dynamics have zero complexity.

```

```

## 2. Five Theorems

### Theorem A — Hidden Permutation Structure

**The operator F: Z/2 × Z/5 → Z/2 × Z/5 is a permutation with:**
- **Four fixed points:** (0,0), (0,3), (1,3), (1,4)
- **One 6-cycle:** (1,1) → (1,2) → (0,1) → (1,0) → (0,4) → (0,2) → (1,1)

*Proof.* The transition table (Section 4) defines F explicitly. Direct inspection
confirms 4 fixed points and one cycle of length 6. Bijectivity follows from
injectivity on a finite set. □

### Theorem B — Exact ε' Rule

**The parity component satisfies:**

```
ε'(ε, y) = (1−ε)·𝟙[y∈{1,2}] + ε·𝟙[y∈{1,3,4}]
```

Equivalently:

| y    | ε' rule  |
|------|----------|
| y=0  | reset    |
| y=1  | set      |
| y=2  | flip     |
| y∈{3,4} | preserve |

*Proof.* Direct computation on all 10 states. Verified. □

### Theorem C — Exact y' Rule

**The mod-5 component satisfies:**

```
y'(ε,y) = (1−ε)·P₀(y) + ε·P₁(y)  (mod 5)

P₀(y) = 3y + 2y² + y³ + 4y⁴      (mod 5)
P₁(y) = 4 + 3y + 3y² + y³ + y⁴   (mod 5)
```

*Proof.* Any function Z/5 → Z/5 has a unique polynomial representation of degree ≤ 4.
The two branches are the Lagrange interpolants through the required values. Direct
verification on all 10 states. □

### Theorem D — Unique Minimal Split Operator

**The operator F = (F_ε, F_y) from Theorems B and C is the unique minimal
split operator on Z/2 × Z/5 compatible with the morphotic braid.**

*Proof.* Uniqueness: F is determined by its values on 10 points. The transition
table specifies all 10 values. Minimality: no simpler unified polynomial
(Families A–C tested) fits exactly. The piecewise split on ε is the minimum
required complexity. □

### Theorem E — Conjugacy to Rotation + Identity

**The operator F is conjugate to:**
- k → k+1 (mod 6) on the 6-cycle states
- identity on the 4 anchor states

*Proof.* Define the cycle index k by k=0:(1,1), k=1:(1,2), k=2:(0,1), k=3:(1,0),
k=4:(0,4), k=5:(0,2). Verify that F maps each cycle state to the next index.
Verify the 4 anchors are fixed. The conjugacy φ is the cycle index map. □

**Consequence.** The split operator's apparent complexity is the coordinate
cost of Z/2 × Z/5 encoding, not dynamical complexity.

## 3. Wobble and Propagation

### 3.1 Unit Coherence Propagation

Define the braid coordinate β: {0,...,9} → {0,...,9} by β(braid[i]) = i.

**The braid is a unit-rate traversal: Δβ/Δn = 1 for all steps.**

Direct computation: Δβ = [1,1,1,1,1,1,1,1,1] throughout. The "speed limit"
of the system is not imposed — it is the geometry of the permutation.

### 3.2 Wobble Constant Derived

The 6-cycle occupies 6 of 10 visible states. The cycle occupancy fraction is 6/10 = 3/5.

```
W_BHML = cycle_occupancy / ring_size = (3/5) / 10 = 3/50
```

The wobble constant W_BHML = 3/50, previously derived from cross-cycle disagreement
in Z/10Z arithmetic, is now identified as the **cycle occupancy fraction normalized
by the ring size** — the fraction of the ring in propagating mode.

## 4. The π-Seed as Partial Witness

The sequence 071 (the first three digits of the braid) is witnessed by the
π-repeat ladder:

| Depth | First repeat of 3.14159... | Position    | Following digit |
|-------|---------------------------|-------------|-----------------|
| 8     | "31415926"                | 50,366,472  | **0**           |
| 6     | "314159"                  | 176,451     | **7**           |
| 5     | "31415"                   | 88,008      | **1**           |

The seed 071 is not arbitrary. It witnesses:
- **0**: the fixed anchor
- **7**: the cycle entry point
- **1**: the first σ⁻¹ step from 7

Beyond three slots: the digit 6 does not appear as a following digit of any
5+, 6+, 7+, or 8-digit π-prefix repeat within 5 million tested digits.
The full braid is not recoverable from π-repeats by any deterministic rule.

**Conclusion.** The π-repeat ladder is a partial external witness of the first
three braid slots. The Prog channel is the primary algebraic source.
