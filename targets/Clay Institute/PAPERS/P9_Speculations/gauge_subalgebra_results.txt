2. THE ALTERNATING STRUCTURE: skew <-> sym
========================================================================

Level 0: T, B (symmetric)
Level 1: C = [T,B] (skew-symmetric)
Level 2: [C,T], [C,B] (symmetric)
Level 3: [[C,T],B], [[C,B],T], [[C,T],[C,B]] (mixed)

Commutator type rules:
  [sym, sym] -> skew
  [skew, skew] -> skew
  [skew, sym] -> sym
  [sym, skew] -> sym

So the Lie algebra decomposes as:
  gl(10,R) = sym(10) + skew(10)
  dim(sym) = 10*11/2 = 55
  dim(skew) = 10*9/2 = 45
  Total: 55 + 45 = 100 = dim(gl(10,R))

The SKEW part (so(10), dim=45) is a LIE SUBALGEBRA:
  [skew, skew] = skew (closed under commutation)

The SYM part is NOT a subalgebra:
  [sym, sym] = skew (NOT sym)

This is a CARTAN DECOMPOSITION:
  gl(10,R) = so(10) + sym(10)
  so(10) = maximal compact subalgebra
  sym(10) = non-compact complement
