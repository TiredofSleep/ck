# Bible App — Separate Target

This is a **separate application** from CK the coherence enforcer.

CK's TIG algebra can be applied to text (D2 curvature on Hebrew/Greek roots),
but that is a distinct use case from OS steering. These targets do not share
a runtime.

## What goes here

- Bible study tools using D2 force vectors on scripture text
- Hebrew root → 5D force vector analysis
- Separate launch, separate process, separate purpose

## Status

Planned. Not yet implemented as standalone target.
The ck_claude_library.py Bible functions currently live in ck_sim/doing/
and should eventually be extracted here.
