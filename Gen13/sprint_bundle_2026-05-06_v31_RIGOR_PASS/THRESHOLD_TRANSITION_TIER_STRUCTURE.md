```

Most previously-labeled misses are actually **Tier 2 matches** — they fit clean composites once you accept composite forms.

## 5. The unfalsifiability concern (honest)

Adding Tier 2 (composites of Tier 1) dramatically expands the matching vocabulary. Almost any small rational number can fit some (k·m) ± d / N^p form once Tier 2 is allowed.

**This is the unfalsifiability problem the framework needs to address.**

If the framework is real, Tier 2 elements should:
1. **Recur across multiple unrelated domains** (like 73 appearing in HARMONY count AND Bohr radius)
2. **Not require ad-hoc form changes** (the same composite shouldn't be patched differently in different contexts)
3. **Match better than random Tier 2 vocabulary would** (this requires null testing)

The (k·m) ± d pattern observed here is suggestive but not statistically discriminating because most small primes can be expressed this way.

**The honest scientific test** is the random null test: 
- Generate 200 random rationals from Tier 1 + Tier 2 vocabulary
- Test how many fit arbitrary physical constants within W_BHML band
- If TIG's hit rate is comparable, the matches are accidental
- If TIG's hit rate is significantly higher, there's signal

This null test has not yet been run.

## 6. What the threshold structure suggests

Despite the unfalsifiability concern, the data structure DOES exhibit features:

```
1. Tier 1 saturates around small primes 2-9
2. Tier 2 emerges naturally with composites and derived counts (73, 17, 22)
3. Tier 3 transcendentals are honestly out of scope (irrational by theorem)
4. Specific physical phenomena (atomic edge cases, helium ionization) may
   require additional physics-tier elements

The threshold transition between Tier 1 and Tier 2 isn't arbitrary — Tier 2 elements derive from Tier 1 structure (skeleton = 22 = TSML pre-structure cells, HARMONY count = 73 = TSML harmony cells). They're "compositions of the substrate's own structure," not random additions.

**This is the structural feature worth examining further.** Not "do small composites match physical constants" (most small composites match many things), but "do the same composites recur in multiple unrelated contexts."

That recurrence pattern is what the framework should emphasize when it goes to scrutiny phase.
