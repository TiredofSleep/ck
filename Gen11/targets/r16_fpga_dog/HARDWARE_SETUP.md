## Bitstream

Flash `Gen9/targets/zynq7020/build/ck_full.bit` to microSD as BOOT.BIN.

```
BOOT.BIN components:
  - FSBL (generated by Vivado from ck_build_full.tcl)
  - Bitstream: ck_full.bit
  - ARM application: ck_brain.elf (from Gen9/targets/fpga/arm/)
```

Full build: `source Gen9/targets/zynq7020/build/ck_build_full.tcl` in Vivado.

## Wiring

### FPGA to Servo Bus

```
FPGA JM2 Pin 3 (G19) --- UART TX ---> Dog servo controller RX
Dog servo controller TX --> UART RX --> PS UART1 RX (MIO pin)
FPGA GND                ------------> Dog GND (shared)
```

Level shifter between FPGA 3.3V and servo bus 5V if needed.
XiaoR servo controller is usually accessible by removing the back panel.

### Servo IDs

```
Leg       Hip ID   Knee ID
FL (0)      1         2
FR (1)      3         4
BL (2)      5         6
BR (3)      7         8
```

Center position: 500ms pulse / position 500 (0-1000 range).

## Software Architecture

```
R16 (CK engine)
  ↓  USB serial (115200 baud)
FPGA (Zynq-7020)
  ↓  ck_protocol.py binary packets
  ├── BEING: 50Hz CL heartbeat → coherence
  ├── DOING: D2 pipeline → operator → gait_vortex
  └── UART TX → servo bus
            ↓
    XiaoR dog legs move
```

---

## Phase → Gait Mapping

| Condition | Gait | Description |
|-----------|------|-------------|
| C < 0.20 | E-STOP | Coherence too low — servo center |
| Phase 1 (λ < 0.09) | STAND | Field settling, no motion |
| Phase 2 (0.09 ≤ λ < 0.50) | WALK | Building momentum |
| Phase 3 (λ ≥ 0.50) | TROT | Full doing energy |

## Bring-Up Sequence

```
Step 1: Flash BOOT.BIN to microSD, insert in FPGA, power on.
        Verify: heartbeat LED blinks (LED0, R19)

Step 2: Connect USB cable R16 ↔ FPGA.
        Find COM port: Device Manager → Ports (look for "USB Serial")

Step 3: Leash test (no-servo, protocol only):
        python ck_leash_test.py COM3 --verbose --no-servo

Step 4: Leash test (with dog on bench, tethered):
        python ck_leash_test.py COM3 --verbose

Step 5: Bridge (live control):
        python ck_r16_bridge.py --port COM3

Step 6: Full launch:
        LAUNCH_DOG.bat
```

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| No PONG on ping | FPGA not booted / wrong COM | Check LED, re-flash BOOT.BIN |
| Tick rate = 0 Hz | ARM firmware not running | Check BOOT.BIN includes ck_brain.elf |
| Servos don't move | Wiring, level shifter | Check JM2 Pin 3, GND, voltage |
| ESTOP immediately | Coherence < 0.20 on cold boot | Wait 30s for field to settle |
| Word soup in voice | Fractal voice cold start | Allow 100k+ heartbeat ticks |
