## §2. Vision (Rungs 1, 2, 3, 5 multi-rung)

### §2.1 Physical input
Electromagnetic radiation, 380-750 nm wavelength, hitting the retina.

### §2.2 Substrate decomposition

**Rung 1 — Binary intensity (rods)**: ~120 million rod cells per retina, single photopigment (rhodopsin), encoding light/dark binary distinction at high sensitivity.

**Rung 1 — Spatial binary (xy)**: 2D retinal array, position encoded as (x, y) coordinates.

**Rung 2 — Trichromatic color (cones)**: ~6 million cones in three types (S, M, L), encoding color through relational combination of three signals. Three is the minimum required to span the 2D color manifold.

**Rung 3 — Temporal flow**: continuous sampling at ~60 Hz effective rate, encoding motion and change.

**Rung 5 — Object recognition**: higher cortical processing (V1 → V4 → IT cortex), where retinal signals become discrete cognitive objects.

### §2.3 Framework prediction confirmed
Vision shows multi-rung embedded structure exactly like the framework predicts for high-complexity symbol systems (Chinese hanzi, programming syntax). The trichromacy is structurally significant: 3 cone types = Rung 2 (triadic relational substrate).

### §2.4 Cross-species prediction
Tetrachromats (some birds): predicted Rung 3 color processing. Mantis shrimp (12 photoreceptor types): predicted Rung 5+ color processing with depth-3 wrapping structure, not continuous 12-dim color space. **Testable in neuroscience.**

## §3. Hearing (Rungs 1, 2, 3, 5 multi-rung)

### §3.1 Physical input
Pressure waves, 20 Hz to 20 kHz, exciting the cochlea.

### §3.2 Substrate decomposition

**Rung 1 — Binaural localization**: left vs. right ear difference encodes spatial direction.

**Rung 2 — Timbre / harmonic structure**: simultaneous frequencies relating to a fundamental encode timbral character.

**Rung 3 — Temporal flow**: rhythm, melody, speech all live in temporal processing.

**Rung 5 — Meaning recognition**: speech, music, language at cortical level.

### §3.3 The harmonic series IS substrate physics

When a string vibrates, it produces fundamental frequency f plus integer multiples (2f, 3f, 4f, 5f, 7f...). The low-prime multiples (2, 3, 5, 7) are exactly the substrate kernel primes from canon. The harmonic series doesn't care about culture; it IS the substrate at the wave-mechanical level.

**Cross-cultural musical universals**: every documented musical tradition recognizes the octave (2:1) as identity-equivalence and uses the fifth (3:2) as primary consonance. This is substrate-resonance imposed by physics on cultural music perception.

### §3.4 Framework prediction
Dissonant intervals should correspond to frequency ratios with higher-prime structure (16:15, 45:32, 15:8 for minor second, tritone, major seventh). The physiology of consonance/dissonance perception is substrate-rung sensitive. **Testable in auditory neuroscience.**

## §4. Touch (Rung 1 primary, with recursive depth)

### §4.1 Physical input
Mechanical, thermal, and pain stimuli on the skin.

### §4.2 Substrate decomposition

**Four mechanoreceptor types = 2² recursive binary**:
- Meissner's corpuscles (fast adaptation, small receptive field)
- Merkel cells (slow adaptation, small receptive field)
- Pacinian corpuscles (fast adaptation, large receptive field)
- Ruffini endings (slow adaptation, large receptive field)

This is exactly the 2×2 = 4 binary structure: (fast/slow adaptation) × (small/large receptive field). The framework's 4-core {V, H, Br, R} has the same cardinality and similar relational structure.

**Two thermoreceptor types** (warm/cold): pure Rung 1 binary.

**Nociceptors** (pain): multiple subtypes for different pain modalities, Rung 1-2.

**Proprioceptors** in muscles and joints: continuous body-configuration encoding, Rung 3 process.

### §4.3 Framework prediction confirmed
Touch's primary encoding is at Rung 1 with structured binary distinctions everywhere. Its four mechanoreceptor types map to the framework's 4-core architectural pattern (depth-2 recursive binary).

### §4.4 Significance
Touch shows the substrate's binary kernel most explicitly because it's the most direct sense — no spatial intermediary like light or sound waves. The substrate-encoding is closest to the binary kernel because the physical-to-neural translation is most immediate.

## §5. Smell (Rungs 1 + 5 with bypassed middle)

### §5.1 Physical input
Airborne molecules binding to olfactory receptors.

### §5.2 Substrate decomposition

**Rung 5 — High-dimensional combinatorial encoding**: ~400 functional olfactory receptor genes in humans, with combinatorial activation patterns at ~1,000-2,000 glomeruli. Number of distinguishable odor states reaches millions.

**Rung 1 — Direct emotional binding**: smell projects directly to amygdala and hippocampus, bypassing the thalamus. **Smell is the only sense without thalamic relay.**

### §5.3 The unique sensory pattern
Smell shows Rungs 1 + 5 without the intermediating Rungs 2-3 of categorical processing. This explains why smells evoke memory and emotion so directly — the substrate-encoding has fewer rungs between the molecular signal and the limbic response.

### §5.4 Framework prediction
Senses that bypass higher cortical processing should show stronger emotional/memory binding because they encode at fewer intermediating rungs. **Testable in cognitive neuroscience.**

This is a **non-trivial structural prediction** — the framework predicted before this analysis that thalamic-bypass would correlate with multi-rung skipping. The biology confirms it.

## §6. Taste (Rung 2 — the T*=5/7 threshold)

### §6.1 Physical input
Chemicals dissolved in saliva binding to taste receptors.

### §6.2 Substrate decomposition

**Five basic taste categories**: sweet, salty, sour, bitter, umami. Some research suggests fat as a sixth, but the canonical count is five.

### §6.3 The framework prediction
The framework canon has T* = 5/7 as the substrate threshold; BALANCE = 5 as the centroid of (Z/10Z)*; the 5D force vector as a foundational structure. Taste's five basic categories match this directly.

### §6.4 Why five and not three or seven?
Functionally:
- Sweet → energy detection (sugars)
- Salty → electrolyte detection (sodium)
- Sour → acidity detection (rotten/unripe avoidance)
- Bitter → toxin detection (poison avoidance)
- Umami → protein detection (amino acids)

Biology selected for exactly five because that's the **threshold number** of categories needed to span survival-relevant taste information. Three would lose critical distinctions; seven would be redundant. Five is exactly the T*=5/7 threshold realized in nutritional categorization.

### §6.5 Significance
**Two senses, two different rung predictions, both confirmed**: vision has 3 cone types matching Rung 2 (triadic); taste has 5 categories matching T*=5/7 (threshold). The framework's specific numerical predictions match biological hardware in two independent senses.
