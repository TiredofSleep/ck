## §2. Six observational rules extracted from the tests

These are derived directly from the test results and are Tier B-structural (each is supported by multiple tests).

### Rule 1 — Rung determines component count

Symbols decompose into a number of structural components determined by their substrate rung:
- **Rung 0**: 1 component (pure boundary)
- **Rung 1**: 2 components (binary)
- **Rung 2**: 3 components (relational triadic)
- **Rung 3**: 3 primary components with internal Rung 2 substructure
- **Rung 5**: 3 components each with rich internal substructure

**Supported by**: digit tests (uniformly 3-component at Rung 5), letter tests (predominantly 2-component at Rung 1), math operators (3-component at Rung 2), Hangul vowels (3-component triadic).

### Rule 2 — Three components map to kernel + strand + wrapping

When 3-component decomposition occurs (Rung 2 and above), the components correspond to:
- **Kernel**: primary structural element (vertical stroke, closed loop, etc.)
- **Strand**: secondary or connecting element
- **Wrapping**: spin, join, or void that completes the encoding

This maps directly to the Braiding Fractal architectural template. Every substrate-resonant symbol IS a miniature Braiding Fractal.

**Supported by**: digit decompositions (every digit shows kernel-strand-wrapping structure), Hangul consonants (articulator + place + relation), math operators (operand + operand + relation).

### Rule 3 — Higher rungs embed lower rungs

When symbols contain internal structure, higher-rung primary structures contain lower-rung secondary structures. Embedding direction is always higher → lower (never the reverse).

- Rung 5 symbols (digits) contain Rung 2 substructure within components
- Rung 3 symbols (music notation) contain Rung 1 substructure
- Latin alphabet's Rung 1 contains Rung 0 embedded (unary letters C, O, S, U)

**Supported by**: hanzi multi-rung embedding (composition at Rung 1, radicals at Rung 3), music notation (Rung 3 primary with Rung 1 dynamics embedded), programming syntax (Rung 2 operations + Rung 1 control flow).

### Rule 4 — System-level substrate-resonance can occur without unit-level resonance

The substrate can appear at the *organizational* level of a symbol system even when individual symbols are conventional.

**Supported by**: periodic table (element symbols are conventional Latin/Greek abbreviations; periodic table organization is substrate-resonant at Rung 5).

### Rule 5 — Substrate-resonance is selected over historical time

Symbol systems that match substrate properties are culturally durable; symbol systems that don't get displaced.

**Supported by**: Cuneiform numerals (pre-substrate-resonant tally encoding, historically replaced by Indo-Arabic positional notation); the historical chain Brahmi → Gupta → Nagari → Arabic → Modern numerals showing convergent evolution toward 3-component decomposition.

### Rule 6 — Deliberate design lands at substrate-appropriate rungs

When humans design symbol systems from first principles for cognitive accessibility, they naturally encode at substrate-resonant rungs.

**Supported by**: Hangul (deliberately designed in 1443 for articulatory diagramming, lands at Rung 2 as framework predicts); programming syntax (designed by engineers for cognitive accessibility, lands at Rungs 1 and 2 as framework predicts).

## §3. Attempted compression: three unified rules

The six observational rules can be compressed into three structural rules.

### Rule A — Form is determined by rung

A symbol at Rung k has structural form determined by depth-(min(k, 3)) wrapping. At k=0, this is 1-component. At k=1, this is 2-component. At k≥2, this is 3-component with internal structure scaling with k.

**Subsumes**: Rules 1 and 2.

### Rule B — Hierarchical embedding

Substrate rungs embed hierarchically. Higher-rung structures contain lower-rung structures. Substrate-resonance can appear at any level of the hierarchy independently — at the unit level, the system level, or both.

**Subsumes**: Rules 3 and 4.

### Rule C — Substrate-resonance is convergent

Substrate-resonance is reached through selection (in evolved systems) or choice (in designed systems). Either pathway produces substrate-resonant systems; substrate-non-resonant systems are unstable.

**Subsumes**: Rules 5 and 6.

**Status of A/B/C**: Tier B-exploratory. The compression is legitimate but the unified rules are slightly more abstract than the observational rules — they lose specificity in gaining generality.

## §4. Attempted further compression: single Rule R

The three unified rules can be further compressed into a single statement.

### Rule R — Substrate as information equilibrium

**Substrate rungs are the equilibrium attractors of organized information. Any system that organizes information will converge toward substrate-rung structure under appropriate selection pressure.**

**This subsumes A, B, and C** as follows:
- A follows because equilibrium structure has specific form (depth-3 wrapping)
- B follows because equilibrium structures embed hierarchically (smaller equilibria within larger ones)
- C follows because equilibrium is reached through selection or design pressure

**Status of Rule R**: Tier C-exploratory. The compression is theoretically appealing but does not, on its own, generate new predictions that the unpacked rules don't generate. The work is still done by the unpacked rules; Rule R is a label, not a generator.

**Honest assessment**: Rule R feels like deep insight because it compresses many observations into one sentence. But the compression is *post hoc*. To use Rule R, one must unpack it back into A, B, C and then into the six observational rules. The deepest defensible statement remains the six observational rules.

## §5. Attempted extension to non-symbol domains

If Rule R is correct, it should generate predictions for non-symbol domains where information is organized.

**This is where the work gets thinner. I want to be honest about the limits.**

### §5.1 Biology

The framework's canon already states:
- DNA: A-T = 100% TSML harmony, G-C = 50%, ATG = only dual-coherent codon
- 20 amino acids = 5 × 4 force-structure crossings
- GC/AT ratio ≈ T*
- Helix pitch 10.5 = 21/2 = (3·7)/2

These are **canonical** framework claims. They were derived from substrate properties and match observed biology.

But asking whether the framework "predicts" DNA codons being 3 bases is **retrofitting**, not predicting. The 3-base codon structure is a known fact; the framework can be consistent with it without having predicted it independently.

**Honest status**: biology shows substrate-correlations that are already in canon. Whether these constitute *predictions* of the framework or *consistencies* with observed biology is unclear without rigorous independent derivation.

### §5.2 Music

**Honest assessment**: musical traditions vary enormously. The Western 7-note diatonic scale matches HARMONY=7, but:
- Indian classical music uses 22 microtonal divisions of the octave (no clean match to substrate)
- Arabic music uses quarter-tone scales (also no clean match)
- Pentatonic scales (common across many cultures) use 5 notes (could be threshold T*=5/7)
- Pythagorean tuning gives clean ratios (octave 2:1, fifth 3:2) that match substrate primes

The cleaner observation is that **physical wave mechanics produces the harmonic series** (integer-multiple frequencies), which is a substrate-like structure imposed by physics. Cultural variation in how this gets organized into scales is selection within substrate constraints.

**Honest status**: music is partially substrate-resonant in physics-imposed ways (harmonic series), but cultural scale systems are diverse. The framework can be consistent with music but doesn't uniquely predict any particular musical structure.

### §5.3 Linguistics

**Honest assessment**: linguistic universals exist (Chomsky's Universal Grammar program identified many) but cross-linguistic variation is also enormous.

- Subject-verb-object as default order: actually SOV is more common cross-linguistically than SVO; only some languages (English, Mandarin, French) are SVO-default
- Tense system: most languages have tense distinctions but many (Mandarin, Indonesian) primarily use aspect rather than tense
- Phrase structure recursion: genuinely universal across known languages, supporting the Rung 3+ embedding prediction

**Honest status**: linguistics shows some substrate-correlations (recursion, hierarchy) but the specifics vary too much to claim the framework predicts particular linguistic structures.

### §5.4 The honest summary

The framework's strongest predictions are for **deliberately designed symbol systems** (Hangul, programming syntax) where the prediction matches the design choice precisely.

For **evolved symbol systems** (alphabets, numerals), the framework's predictions match observed structure with moderate strength.

For **biology, music, linguistics**, the framework can be made *consistent with* observed structure but does not uniquely predict it. The fit is real but it's partly retrofitting, partly canonical (already in framework), and partly genuine.

**This is normal for an exploratory research program.** What matters is that the strongest predictions (deliberately designed systems) hold, and the methodology is testable for any new domain.

## §6. What this means for the framework

### §6.1 The strong claim that survives

**Symbol systems that are deliberately designed for cognitive accessibility encode at substrate-resonant rungs determined by the system's primary domain of distinction.** This is supported by Hangul, programming syntax, and (partially) by the historical convergence of numeral systems.

### §6.2 The moderate claim that survives

**Evolved symbol systems converge toward substrate-resonance over historical time through cultural selection.** This is supported by the historical-evolutionary findings (Cuneiform displacement, Brahmi-to-Arabic numeral evolution).

### §6.3 The weak/exploratory claim

**The substrate's organizational principles apply beyond symbol systems to any organized information.** This is a theoretical extension that needs domain-specific testing before it can be load-bearing.

### §6.4 What's not justified

- Claiming the framework predicts all organized information
- Claiming Rule R is the single deepest law
- Claiming biology, music, and linguistics are substrate-resonant in detail without rigorous independent derivation

## §7. The methodology going forward

Future tests should:

1. **Make predictions before analyzing** — record what the framework predicts for an unstudied symbol system before checking the actual structure
2. **Verify against primary sources** — every glyph description and structural claim must be checked against scholarly references
3. **Maintain tier discipline** — distinguish strong claims (deliberately designed systems) from moderate claims (evolved systems) from exploratory claims (non-symbol domains)
4. **Track failures explicitly** — when predictions don't match, document it. The framework's credibility depends on owning failures.
5. **Resist the master-key temptation** — the impulse to compress everything into a single deep law is structurally similar to the impulse that takes good frameworks into pseudoscience. The unpacked observational rules are where the real work lives.
