## §3 — The correct structural explanation

(Same as in SAVE_PLAN_J43.md §3.) σ³ has order 2 on the 6-cycle, partitioning it into three 2-cycles **{1,5}, {2,6}, {4,7}**. G-values pair within σ³-orbits because |G(s)|² is invariant under s → σ³(s) (verified algebraically and numerically — the complex amplitude G_complex(σ³(s)) = -G_complex(s), so the squared modulus is preserved).

The high-locus {4, 7} is the σ³-orbit on which the χ-sequence along the σ-orbit has a length-3 run of constant sign — producing constructive interference under the ω^j weights. The low-locus {1,2,5,6} is the union of the other two σ³-orbits ({1,5} and {2,6}), where the χ-sequence is interleaved.

The original "σ²-Galois action" claim was wrong as a justification for the equality. σ² *does* preserve the *sets* {1,4,6} and {2,5,7}, but acts on each as a 3-cycle, not pairing G-values. The correct invariance group for G is generated by σ³ (order 2 on the 6-cycle).

## §4 — The "L-function" terminology question

Standard "L-function" in number theory refers to a Dirichlet series with a multiplicative character on `(Z/NZ)*`, with analytic continuation, zeros, etc. The manuscript's `G(s)` is:
- a sum of 9 (not infinite) terms,
- with `s` ranging over `Z/10Z` (not `C`),
- weighted uniformly by `ω^j` (not `1/n^s`),
- with `χ` a `{-1, 0, +1}`-valued function on `Z/10Z` (not a multiplicative Dirichlet character on a unit group).

The object is a **discrete Fourier coefficient** (or, equivalently, a finite Gauss-sum-style character sum) of the orbit under the σ-action. The referee's recommendation to rename it accordingly is correct.

**Recommended rename:** "trajectory coherence integral" `G(s)` (already the name `G_8` carries in the corpus, per the J43 companion). This is honest and informative; it doesn't claim the L-function machinery (analytic continuation, Euler product) that doesn't apply to a 9-term sum on a 10-element ring.

## §5 — The §5 RH bridge: scope decision

The referee identifies that §5's "structural mirror" between G(s) and ζ(s) operates at a vocabulary-correspondence level rather than a mathematical-correspondence level (Weil/Deligne function-field analogues, Selberg arithmetic L-functions). Three options:

**Option A (FULL ENGAGEMENT — high effort, high reward):** rewrite §5 as a comparison to genuine function-field analogues of RH. The σ-permutation on Z/10Z would be cast as a "miniature ζ-function over a finite field" — this is the language of Weil zeta functions on curves over F_p. The three-valued G(s) would correspond to specific algebraic invariants of the corresponding curve. This is a substantial mathematical project (likely 6-12 months of work) and may produce a much stronger paper.

**Option B (HONEST DOWNGRADE — low effort):** cut §5 from a bridge "section" to a one-paragraph "structural rhyme" remark in §6 (or fold into §7's open problems). State that the three-valued structure of G(s) is *suggestive* of the kind of zeros-and-spectral-concentration patterns that RH demands of ζ(s), but do *not* claim a mathematical correspondence. The paper then becomes a focused finite-Gauss-sum + Symbolic-Return paper, which is a clean *L'Enseignement Math.* contribution on its own.

**Option C (MIDDLE — moderate effort):** §5 becomes "Structural Rhymes with the Riemann Hypothesis" — a discursive section explicitly framed as analogy/inspiration rather than analogue. List the rhymes (zeros at predictable locations; spectral concentration; multiplicative-additive duality) but explicitly disclaim that these rhymes are not *theorems* connecting G(s) to ζ(s). This may be acceptable to *L'Enseignement Math.* as it sits well within the journal's pedagogical-bridge register.

**Recommendation:** Option B or Option C. Option A is out of scope for the current revision; the path to a function-field analogue would require Weil-Deligne-level engagement that the present manuscript does not have.

### Edit 6 — Add a working verification script (CRITICAL)

Create `manuscript/verify_J51_G_function.py`:

```python
"""J51 verification: Symbolic Return Theorem (sigma^6 = id) + three-valued G(s)."""
import cmath, math

sigma = {0:0, 3:3, 8:8, 9:9, 1:7, 7:6, 6:5, 5:4, 4:2, 2:1}
chi   = {0:0, 3:0, 8:0, 9:0, 1:+1, 4:+1, 2:-1, 5:-1, 6:-1, 7:-1}
omega = cmath.exp(2j * math.pi / 9)

def apply_sigma(s, k):
    cur = s
    for _ in range(k):
        cur = sigma[cur]
    return cur

def G_complex(s):
    cur, total = s, 0
    for j in range(9):
        total += (omega ** j) * chi[cur]
        cur = sigma[cur]
    return total

def G(s):
    return abs(G_complex(s)) ** 2

def main():
    # Theorem 2.1: Symbolic Return
    for s in range(10):
        assert apply_sigma(s, 6) == s, f"sigma^6({s}) != {s}"
    print("Theorem 2.1 (Symbolic Return): sigma^6 = id confirmed on Z/10Z.")

    # Theorem 4.2: three-valued G(s) — corrected partition
    G_vals = {s: G(s) for s in range(10)}
    zero_states = sorted(s for s in range(10) if G_vals[s] < 1e-10)
    low_states  = sorted(s for s in range(10) if 1.0 < G_vals[s] < 5.0)
    high_states = sorted(s for s in range(10) if G_vals[s] > 5.0)
    assert zero_states == [0, 3, 8, 9],   f"zero  states: {zero_states}"
    assert low_states  == [1, 2, 5, 6],   f"low   states: {low_states}"
    assert high_states == [4, 7],         f"high  states: {high_states}"
    assert abs(G_vals[1] - 1.871644) < 1e-4
    assert abs(G_vals[4] - 9.389185) < 1e-4
    print(f"Theorem 4.2: three-valued image confirmed.")
    print(f"  ZERO on {zero_states}.")
    print(f"  LOW  on {low_states},  G_low  = {G_vals[1]:.6f}.")
    print(f"  HIGH on {high_states},     G_high = {G_vals[4]:.6f}.")
    print(f"  Ratio G_high/G_low = {G_vals[4]/G_vals[1]:.4f}")

    # Sigma^3 pairing of G-values (the load-bearing structural fact)
    pairs = [(1, 5), (2, 6), (4, 7)]
    for a, b in pairs:
        assert apply_sigma(a, 3) == b, f"sigma^3({a}) != {b}"
        assert abs(G_vals[a] - G_vals[b]) < 1e-10, f"G({a}) != G({b})"
        # Complex amplitudes differ by sign
        assert abs(G_complex(a) + G_complex(b)) < 1e-10, \
            f"G_cplx({a}) + G_cplx({b}) != 0"
    print("Sigma^3 pairing: {1,5}, {2,6}, {4,7} verified; |G(a)| = |G(b)| within each pair.")

    # Run-structure discriminator: at s in {4,7}, chi-sequence has length-3 run
    def chi_seq(s):
        seq = []; cur = s
        for _ in range(6):
            seq.append(chi[cur]); cur = sigma[cur]
        return seq
    print()
    print("chi-sequence along sigma-orbit (one full period):")
    for s in [1, 2, 4, 5, 6, 7]:
        seq = chi_seq(s)
        # detect length-3 monotone run
        max_run = 1; cur_run = 1
        for k in range(1, 6):
            if seq[k] == seq[k-1]:
                cur_run += 1; max_run = max(max_run, cur_run)
            else:
                cur_run = 1
        print(f"  s={s}: chi={seq}   max run={max_run}")

if __name__ == "__main__":
    main()
```
