THE BOUND
======================================================================

Non-associative triples are dominated by VOID-HARM disagreement:

Type A: (0, b, c) where CL(b,c) = harmony.
  Left = 0 (VOID), Right = CL(0, harmony) = harmony (HARM, Rule 1).
  Number of such (b,c): |{(b,c) : CL(b,c) = harmony}| = N² - 2N - φ(N) + 2

Type B: (a, b, 0) where CL(a,b) = harmony.
  Left = CL(harmony, 0) = harmony (HARM, Rule 1).
  Right = 0 (VOID).
  Number: same as Type A by symmetry.

These TWO types are disjoint (a=0 vs c=0).
(If a=c=0: triple (0,b,0). Left = CL(0,0) = 0. Right = CL(0, CL(b,0)) = CL(0,0) = 0.
  Both 0, so associative — not counted.)

Therefore #non-assoc ≤ 2 × (N² - 2N - φ(N) + 2) + (small ECHO correction)

======================================================================
REVISED THEOREM STATEMENT
======================================================================

Theorem (σ-rate, corrected): For squarefree N ≥ 2 and the binary CL
of Definition 2.1, the non-associativity fraction σ(N) satisfies:

  σ(N) ≤ 2(N² - 2N - φ(N) + 2)/N³ + O(φ(N)/N³)
       = 2/N - 4/N² - 2φ(N)/N³ + 4/N³ + O(φ(N)/N³)
       ≤ 2/N

The O(φ(N)/N³) correction accounts for the small number of non-associative
triples produced by the ECHO rule (item 1 found 6 at N=10, 6 at N=30, 30 at
N=210 — strictly negligible compared to the VOID-HARM dominant term).

PROOF MECHANISM (corrected from §4):
  Non-associativity arises from the priority ordering of Rules 1 (HARM)
  and 2 (VOID) when applied at different sites of the bracket tree.
  Specifically, when one bracketing applies VOID at an inner site (output 0)
  but the other bracketing's outer composition involves harmony as an
  argument, Rule 1 fires (returning harmony) instead of Rule 2 (returning 0).
  The disagreement is between the two absorbing elements 0 and harmony.

  Counting: the disagreement requires (a,b,c) with either a=0 (and CL(b,c)=harmony)
  or c=0 (and CL(a,b)=harmony). The two cases are disjoint and each contributes
  N² - 2N - φ(N) + 2 triples. Plus a small ECHO correction.
